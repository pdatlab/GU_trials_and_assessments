# FCI #

Functional Comorbidity Index 

# Paper #

- Groll DL, To T, Bombardier C, Wright JG. The development of a comorbidity index with physical function as the outcome. J Clin Epidemiol. 2005;58(6):595–602. 

# Scoring #

Each yes is a point. All points are summed (no weighting). Obesity is scored as yes if BMI exceeds 30 (any other time?). There are 18 questions so the score may vary from 0 to 18 (theoretically).
