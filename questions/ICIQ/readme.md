# ICIQ #

International Consultation on Incontinence Questionnaire

There are a variety of modules.

https://iciq.net/modules



# Scoring #
ICIQ-UI (SF): From 0 to 21

ICIQ-B, "Scores from 1-21 for bowel pattern, 0-28 for bowel control and 0-26 for impact on quality of life associated with anal incontinence symptoms"
